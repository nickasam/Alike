技术栈亮点

  • Vue 3 + Nuxt 3 — SSR/SPA 混合，SEO 友好
  • Go (Gin) — goroutine 高并发，低延迟
  • PostgreSQL — 关系模型 + 全文索引
  • Redis — 会话缓存 + Pub/Sub 广播
  • WebSocket Hub — 房间制实时推送
  • MinIO / S3 — 对象存储，兼容 S3 API
  • Docker Compose — 一键容器化部署

核心数据流

  • REST 写路径 — 前端 → Nginx → Go → PostgreSQL
  • 实时广播 — DB 落库后 publish 到 Redis
  • Pub/Sub 扩散 — 多实例 Hub 订阅同一频道
  • WS 下行 — Hub 推送在线频道成员
  • 共情通知 — 定向推送被共情者
  • 文件旁路 — storage 直传 MinIO 存 URL

特色功能

  • 情绪系统 — 情绪标签 + 频道情绪热力看板
  • 抱团取暖 — 不是"赞"，是"我懂你"
  • 牛马等级 — 活跃/被共情累积自动升级
  • 打工日记 — 每日打卡 + 连续徽章
  • 排行榜 — 最受共情 / 最惨 / 最活跃
  • 匿名模式 — 隐藏身份健康宣泄
  • 线程讨论 — Slack 式回复 + @提及
