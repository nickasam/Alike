# Alike - Web 应用设计文档

## 设计理念

Alike 是一个现代化的 Web 社交平台，专注于为 LGBTQ+ 社区提供安全、友好的交流环境。采用渐进式 Web 应用（PWA）设计理念，提供类似原生应用的体验。

> **文档版本**: 2.1  
> **最后更新**: 2026-03-03  
> **实现状态**: MVP 开发中  
> **注意**: 本文档包含已实现功能和未来规划，请参考状态标识区分

## 设计规范

### 配色方案
```
主色调（渐变系）：
- Primary Gradient: #667eea → #764ba2（紫蓝渐变）
- Accent Gradient: #f093fb → #f5576c（粉红渐变）
- Success: #22c55e（绿色）
- Warning: #f59e0b（橙色）
- Error: #ef4444（红色）

中性色（玻璃态）：
- Glass Background: rgba(255, 255, 255, 0.95)
- Glass Border: rgba(255, 255, 255, 0.2)
- Backdrop Blur: 20px

文字颜色：
- Text Primary: #1a1a1a（深黑）
- Text Secondary: #6b7280（灰色）
- Text Tertiary: #9ca3af（浅灰）

背景颜色：
- Page Background: linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%)
- Card Background: rgba(255, 255, 255, 0.95)
- Input Background: rgba(249, 250, 251, 0.8)
```

### 字体系统
```
字体家族：Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif

标题层级：
- H1: 48px / 900（Hero 标题）
- H2: 32px / 800（页面标题）
- H3: 24px / 700（区块标题）
- H4: 20px / 600（卡片标题）

正文层级：
- Body Large: 18px / 500（重要文本）
- Body: 16px / 500（常规文本）
- Body Small: 14px / 500（次要文本）
- Caption: 13px / 600（辅助文本）

代码/技术：
- Monospace: 'SF Mono', 'Roboto Mono', monospace
```

### 圆角规范
```
超大圆角：28px（Logo/大图标）
大圆角：24px（对话框卡片）
中圆角：16px（按钮/输入框）
小圆角：12px（小按钮/标签）
圆形：50%（头像/状态点）
```

### 间距系统（8px 基准）
```
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
```

### 阴影系统
```
- Shadow SM: 0 4px 16px rgba(0, 0, 0, 0.08)
- Shadow MD: 0 8px 24px rgba(0, 0, 0, 0.12)
- Shadow LG: 0 16px 40px rgba(0, 0, 0, 0.15)
- Shadow XL: 0 24px 60px rgba(0, 0, 0, 0.20)
- Glow: 0 0 40px rgba(240, 147, 251, 0.5)
```
