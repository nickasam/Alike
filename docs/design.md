# Alike App - UI/UX 设计文档

## 设计规范

### 配色方案
```
主色调：
- Primary: #2563EB (深蓝)
- Secondary: #7C3AED (紫色)
- Accent: #EC4899 (粉红)

中性色：
- Background: #F9FAFB
- Surface: #FFFFFF
- Text Primary: #111827
- Text Secondary: #6B7280
- Border: #E5E7EB

功能色：
- Success: #10B981
- Warning: #F59E0B
- Error: #EF4444
```

### 字体系统
```
iOS: SF Pro
Android: Roboto

标题：
- H1: 28px/Bold
- H2: 24px/Semibold
- H3: 20px/Semibold

正文：
- Body: 16px/Regular
- Caption: 14px/Regular
- Small: 12px/Regular
```

### 圆角规范
```
卡片: 16px
按钮: 12px
输入框: 10px
头像: 50% (圆形)
```

## 组件库

### 按钮
```
主要按钮：渐变背景 (#2563EB → #7C3AED)，白色文字，圆角12px
次要按钮：白色背景，蓝色文字，边框
危险按钮：红色背景 (#EF4444)

禁用状态：灰色背景，低透明度
```

```
┌──────────────────────┐
│ 📍 占位符文本        │

```

[运动] [旅行] [美食]

图标 + 文字

## 交互说明

### 手势操作
- **左滑聊天列表**：删除/置顶/标记已读
- **下拉首页**：刷新用户列表
- **长按用户卡片**：快速操作菜单
- **双击图片**：放大查看

### 状态反馈
- **加载**：圆形进度指示器
- **成功**：绿色勾选动画
- **错误**：红色提示 + 抖动动画
- **发送消息**：送达✓️ 已读✓️

### 页面转场
- **Push**：从右侧滑入（详情页）
- **Modal**：从底部弹出（表单）
- **Fade**：淡入淡出（弹窗）
